About

Craigslist Market Analyzer is a MEAN stack web application that I started in March 2016. The goal of the application is to collect data on cars on Craigslist to be able to determine in the future what the true market price of a privately sold vehicle is. While there are various websites like Kelly Blue Book and TrueCar which appraise cars, this app collects real data on Craigslist private listings for any city(currently limited to cities in Texas) on cars sold exclusively by individuals. The main goal of this app is in determining when a good deal pops up on Craigslist.

At its current state, the application is made up of three key components:

Database

The database stores information on the year, make, model, city, and URL of every car that has been saved.

Parser

The parser performs a Craigslist search for a given city, make, model and parses the results to neatly display the data in a table.

Analyzer

The analyzer processes data from the database for a given city, make, model and displays the information in a series of graphs.
